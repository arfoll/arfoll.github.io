Controlling Velux blinds using the KIG 300

2022-09-18
#iot  #homeassistant 
If you have some Velux blinds (SSL, MSL) or windows which use the velux ACTIVE branding you likely can hook them up to control them remotely. In order to do this you need to buy the velux gateway KIG300 which is often sold as a kit with some temp/humidity sensor which I didn’t get. The annoying thing with this setup is that it’s using the apple homekit integration as well as requiring a smartphone app for the initial setup, pairing this with a network that has no internet is also a little tricky, so this is by far not my favourite solution.
